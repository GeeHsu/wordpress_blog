教學一 : 載入地圖

  • 範例程式碼
  • 官網參數
  • 自我練習：換成自己的座標，做成自己的地圖

語法

  • .setView – 布置場景，可以想像成建立一個空的容器地圖
  • tileLayer – 拉近圖資，本教材使用 OpenStreetMap，其他還有像是 mapbox、Google Map，但需要 api key
  • .setZoom(18) – Zoom 地圖
  • getZoom() – 取得 Zoom

為什麼使用別人的服務 API，都會要使用 key？因為要用這個 key 去算你的 $$ 啊~

教學二 : 圖層概念

  • 地圖圖層概念
  • Leaflet 圖層有
    • Raster Layers –
      • TileLayer – 圖磚方式呈現， OpenStreetMap 與 Google Map 都是如此
      • TileLayer.WMS
      • ImageOverlay
      • VideoOverlay
    • UI Layers
      • Marker (單點)
      • Popup (彈跳冒泡視窗 – 可放多內容)
      • Tooltip (彈跳冒泡視窗 – 可放少內容)
    • 其他圖層
      • GeoJSON
      • LayerGroup
      • FeatureGroup
      • GridLayer
    • Vector Layers (向量圖層)
