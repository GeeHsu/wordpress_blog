let、const 差異

let 裡面的 value 是可以被修改、被複寫，
const 是難被修改、被複寫，基本上是不能的。

數值一直在更動適合用 let，
數值比較不常在更動建議使用 const。

回顧：const 無法去做修改，let 是可以被修改。

> const e = 1;
< undefined
> e = 3
x Uncaught TypeError: Assignment to constant variable.
  at <anonymous>:1:2
> let k = 3;
< undefined
> k = 4
< 4
> k
< 4
>
