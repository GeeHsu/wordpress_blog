使用 WordPress、Github架設 Github Pages靜態網站

發表於2019 年 11 月 3 日2019 年 11 月 3 日 由gee.hsu撰寫

準備工具如下:
1.Github – Repository
2.Wordpress – Simply Static
3.Xampp – Apache、MySQL
4.Git Bash

步驟如下:

1.Github 建立一個新的資料庫
資料庫名稱、敘述、公開、勾選初始化資料庫關於 README、建立

2.建立一個新的分支: gh-pages

3.設定 -> 分支
默認分支: gh-pages

4.安裝 Xampp 和 使用phpMyAdmin

5.建立一個本地資料庫

6.下載 WordPress

7.下載 Github 資料庫和解壓縮到 htdocs

8.Wordpress 解壓縮後並移動到對應的資料夾

9.安裝 WordPress

10.需要
phpMyAdmin 新增使用者帳號
勾選全域權限 然後執行
如下:
資料庫名稱、使用者名稱、密碼、資料庫主機位址、資料表前置詞

11.WordPress 安裝網站所需資訊
網站標題、使用者名稱、密碼、電子郵件地址
12.Wordpress Plug-in (外掛)
安裝 Simply Static

13.Simply Static 設定
複製你的網站url 貼在 relative URLs 然後儲存

14.Include / Exclude
增加檔案和目錄
C:\XXX\Xampp\htdocs\XXX/wp-content/

15.Simply Static
Delivey Method change to Local Directory
Local Directory paste on:
C:\XXX\Xampp\htdocs\XXX
然後儲存

16.Simply Static 產生 靜態檔案

17.Git Bash push on Github 資料庫

18.修正某些錯誤
Use absolute URLs
https://xxx.github.io/xxx/

19.Simply Static 產生

20.再一次使用 Git Bash push on Github 資料庫
