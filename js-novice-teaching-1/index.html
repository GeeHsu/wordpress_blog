JS 環境建立與變數

2020 年 9 月 14 日 gee.hsu
  • 學習資源：六角學院直播
  • 變數
  • 協作手冊

開始寫 javaScript、載入 js 環境

<script src="all.js"></script>

js 起手式

alert('hello world!!');

javaScript 主要功能

  1. 可以控制瀏覽器網頁內容
  2. 可以透過 JS，來去跟外部伺服器要資料，或者是更新資料

可以驗證 email 可不可以註冊

變數介紹

MDN – 變量(變數)是值的容器，就像我們可能在總和中使用的數字，或者我們可能用作句子一部分的字符串。

變數名稱值
father09-12345678
girl09-66996699

建立變數、賦值

// 建立一個變數，變數名稱叫做fatherOld，賦予一個值，40
// number 數字
var fatherOld = 40;
var wifeOld = 18;

檢查變數是否建立成功

使用 Chrome 開發人員工具 – Console

fatherOld
40
wifeOld
18

在其他網頁輸入變數名稱是否會回傳變數的值？

會 +1
不會 +2 → 這個

變數的作用範圍

原始型別有哪些

以街景來舉例原始型別內容。

原始型別 (Primitive Type)

  • number 數字
  • string 字串
  • boolean 布林
  • undefined
    • 宣告記憶體、未賦值、var a;
  • null
  • symbol
var fatherName = "tom";
// 字串 string
var fatherOld = 40; 
// 數字 number
var power = false;
// 布林 boolean - true false

在記憶體空間佔位置

宣告記憶體，未賦值， var a;

chrome 是怎麼執行 JS (創造階段、執行階段)

變數重要三劍客：記憶體位置、變數、型別

var 的過程史

  • 建立出執行環境(window、this)
  • 並在跑創造階段 > 執行階段

當 JS 執行時，會有兩個階段

  • 創造階段 (Creation Phase)：設定變數到記憶體
  • 執行階段 (Execution Phase)：逐行執行程式碼

記憶體位置

// 創造階段

console.log(a)
var a = 1;
var b = 2;

console.log(a);

// 1. 紅色錯誤 1
// 2. undefined 1 →這個

// var a;
// var b;
// console.log(a)
// a=1;
// b=2;
