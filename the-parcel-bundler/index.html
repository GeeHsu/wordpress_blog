THE PARCEL BUNDLER

2020 年 6 月 30 日 gee.hsu

A SUPER Easy JavaScript Bundler for your Projects

學習來源：DesingCourse

Parcel

極速、免設定的網頁打包工具。

學習開始

使用 Cmder

  • mkdir parceltest && cd parceltest (建立資料夾和移動到該資料夾)
  • npm init -y (在專案中建立 package.json 檔案)
  • code . (打開程式碼編輯器)

parceltest

  • 建立資料夾：src
    • index.html
    • images/mountain.png
    • styles/main.css
    • index.js
    • bro.js

index.html

<!DOCTYPE html>
<html lang="en zh-hant">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>The Parcel Bundler</title>
    <link rel="stylesheet" href="./styles/main.css">
</head>
<body>
    
    <h1>My page</h1>
    <img src="./images/mountain.jpg" width="300" height="300" alt="my mountain">

    <script src="./index.js"></script>

</body>
</html>

main.css

body {
  background: yellow;
}

bro.js

const bro = (greeting) => {
    return `${greeting}, bro`
}

export { bro }

index.js

import { bro } from './bro'

document.querySelector('h1').textContent = bro(`How's is going`);

Extension – 插件

  • live server

使用 Visual Studio Code 的 Terminal

  • npm i -D parcel-bundler (使用 npm 安裝)

package.json 修改 “scripts”

"scripts": {
  "dev": "parcel src/index.html",
  "prod": "parcel build src/index.html"
},
{
  "name": "parcel",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "dev": "parcel src/index.html",
    "prod": "parcel build src/index.html"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "parcel-bundler": "^1.12.4",
    "sass": "^1.26.9"
  },
  "dependencies": {
    "scrollmagic": "^2.0.7"
  }
}

使用 Visual Studio Code 的 Terminal

  • npm run dev

styles/main.css 修改名稱改為 styles/main.scss

//make a variable
$sup: lightgray;

body {
  background: $sup;
}

index.js 新增 import sass

import { bro } from './bro'
import './styles/main.scss'
document.querySelector('h1').textContent = bro(`How's it going`);

刪除 index.html 的 main.css 載入

<!DOCTYPE html>
<html lang="en zh-hant">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>The Parcel Bundler</title>

</head>
<body>
    
    <h1>My page</h1>
    <img src="./images/mountain.jpg" width="300" height="300" alt="my mountain">

    <script src="./index.js"></script>

</body>
</html>

終止批次工作(在 Terminal 使用 Ctrl + C)，並執行 npm run dev

scrollmagic

  • npm i scrollmagic

import scrollmagic

import { bro } from './bro'
import './styles/main.scss'
import ScrollMagic from 'scrollmagic';

var controller = new ScrollMagic.Controller();

// create a scene
new ScrollMagic.Scene({
        duration: 500, // the scene shoul last for a scroll distance of 100px
        offset: 50 // start this scene after scrolling for 50px
    })
    .setPin("body") // pins the element for the scene's duration
    .addTo(controller); // assign the scene to the controller

document.querySelector('h1').textContent = bro(`How's is going`);

修改 main.scss

$sup: lightgray;

body {
  background: $sup;
  height: 200vh;
}

終止批次工作 ，刪除 dist 資料夾，重新執行 npm run dev，使用 Reveal in file explorer 查看 src.js 的檔案大小

  • 大約 125kb

終止批次工作 ，刪除 dist 資料夾，重新執行 npm run prod，使用 Reveal in file explorer 查看 src.js 的檔案大小

  • 大約 24kb
