Variables

  • let – 用於宣告一個「只作用在當前區塊的變數」，初始值可選擇性的設定。

  • const – 常數不能重複指定值，也不能重複宣告。
  • var – 宣告一個變數, 同時可以非強制性地賦予一初始值。

Hoisting works with function declarations but it doesn’t work with function expressions.
