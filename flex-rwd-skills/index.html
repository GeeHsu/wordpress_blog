Flex 與響應式排版上的技巧

2020 年 5 月 15 日 gee.hsu

共筆文件、影片

簡介

Flex 的基本運作及在響應式網站上的運用技巧

Flex 兩大重點觀念

  • 外容器與內元件
  • 主軸與交錯軸

Flex 需要了解的

  • 軸線觀念
  • 相關屬性 ( 熟記方式 )
  • 外容器：
    • justify-content 是主軸的控制 → Justify 有證明、正當的意思，印刷中有排整齊的意思，所以是主要的軸線內容排整齊的方法。
    • align-items 是交錯軸的控制 → align 校正、對齊的意思，可以想像槍的準心對齊概念，因此是垂直於主軸的排列方法 / items 則是控制 “複數” 的內層元件 ( align-items 複數 vs align-self 單 )
    • flex-wrap 換行的控制 → 如同 CSS 中的 word-wrap，都是可以掌管換行的方式
    • flex-direction 是軸線的控制 → direction 指向的意思
  • 內元件：
    • align-self：交錯軸的單獨元件控制方式　（注意是 self 不是 items )
  • 延伸手法：
    • 與 margin 的配合
      • ex：margin-left: auto;
  • 對齊方法：
    • flex-direction: row | row-reverse | column | column-reverse
    • justify-content: flex-start | flex-end | center | space-between | space-around
    • align-items: flex-start | flex-end | center | baseline | stretch
    • align-self: flex-start | flex-end | center | baseline | stretch
  • BS 使用到的方法

參考資料

  • 圖解：CSS Flex 屬性一點也不難
  • Flex 空間計算規則

RWD 中的基本設定方式

  • 本範例不使用 Bootstrap
  • 請把握三層 container > flex > item
    • container：主要限制最外圍空間使用
      • 限制最大寬
      • 左右保留 padding
    • flex
      • 製作 flex 的層級
      • 請下 flex-wrap
    • item
      • 寬度、margin 設定的地方
      • 內容會在更內層

RWD 中常見的響應式技法

  • 基本排列：左選單右內容
  • 基本排列：多欄式並排，多欄換行
  • 基本排列：置中
  • 桌面版左右至底且數量不定，行動版由上而下：類似 Navbar
  • 桌面版由右至左，行動版由左至右：時間軸
  • 桌面版等高，行動版彈性高度：卡片式排法
  • 相簿排版方法

上面清單最後三個沒有程式碼。

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flex 與響應式排版上的技巧</title>
    <style>
        body {
            margin: 0;
        }
        ol, ul{
            list-style-type: none;
        }

        * {
            box-sizing: border-box;
        }

        .container {
            max-width: 1200px;
            margin: 0 auto;
            padding: 0 10px;
        }

        .d-flex {
            display: flex;
            background-color: #eee;
            /* 響應式排版*/
            flex-wrap: wrap;
        }

        .box {
            height: 200px;
            background-color: hotpink;
            border: 1px solid #333;
        }
        .menu{
            width: 32%;
            margin-right: 3%;
        }
        .content{
            width: 65%;
        }
        .ad{
            width: 32%;
        }
        .hero{
            width: 50%;
        }
        .logo{
            height: 100px;
            width: 100px;
            background-color: #000;
        }

        @media screen and (max-width: 450px) {
            .menu{
                width: 100%;
                margin-right: 0;
            }
            .content{
                width: 100%;
            }
            .ad{
                width: 100%;
            }
            .hero{
                width: 100%;
            }
            .d-flex, .d-flex li{
                width: 100%;
            }
        }
    </style>
</head>

<body>

    <div class="container">

        <div class="d-flex">
            <div class="box menu">
                
            </div>
            <div class="box content">
                
            </div>
        </div>

        <div class="d-flex" style="justify-content: space-between;">
            <div class="box ad"></div>
            <div class="box ad"></div>
            <div class="box ad"></div>
            <div class="box ad"></div>
            <div class="box ad"></div>
            <div class="box ad"></div>
        </div>

        <div class="d-flex" style="justify-content: center;">
            <div class="box hero"></div>
        </div>

        <div class="d-flex" style="align-items: center;">
            <div class="logo"></div>
            <ul class="d-flex" style="margin-left: auto;">
                <li>1</li>
                <li>2</li>
                <li>3</li>
                <li>4</li>
                <li>5</li>
            </ul>
            <a href="#">登入</a>
        </div>

        <div class="d-flex" style="flex-direction: row-reverse;">
            <div class="box menu">
                選單
            </div>
            <div class="box content">
                內容
            </div>
        </div>

    </div>

</body>

</html>
