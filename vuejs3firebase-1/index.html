Single Page Application (SPA)

  • Only a single HTML page sent (initially) to the browser
  • Vue intercepts subsequent requests and handles ‚Äúpage‚Äù changes in the browser by swapping what components are shown on the page
  • Results in a much faster and smoother website experience
