03 – HTML Syntax & Structure

HTML Basics
HTML Syntax & Structure

HTML Syntax

HTML is a Markup Language
We use tags <> to mark content up

Example
<h1>This is a header</h1>

Syntax Summary
* HTML uses tags <> </> to present content
* There are many different tags
* Tags can contain attributes <p class=”myclass”>
* Tags can be nested in other tags
* Tags are normally indented when nested for readability

HTML Basics
Your First Web Page

<html>
 <head>
  <meta charset="utf-8" />
  <title>My first web page</title>
 </head>
 <body>
  <h1>My first web page</h1>
  <p>Yo guys, hope you like my website, we sell fish.</p>
</html>

05 – Head and Body Tag

HTML Basics
The Head and Body

Head and Body tags
* The <head> tag contains all the ‘extra’ information about a web page
– Meta data
– CSS & JavaScript references
* The <body> contains all visual content
– Headings, images, paragraphs etc

<!doctype html>

<html>
 <head>
  <meta charset="utf-8" />
  <meta name="keywords" content="fish, smelly, trout, salmon, shark">
  <meta name="description" content="We sell the smelliest fish online, guranteed!">
  <title>Mr Green's Smelly Fish Emporium</title>
  <link rel="stylesheet" type="text/css" href="css/main.css">
 </head>
 <body>
  <h1>My first web page</h1>
  <p>Yo guys, hope you like my website, we sell fish.</p>
 </body>

</html>

10 – HTML Links

The Anchor Tag

  • <a href=”link-source”>
  • Internal – links to pages in own website
  • External – links to other websites
  • Downloads – links to files
  • Anchors – links to specific areas on a page
<a href="contact.html">Contact Us</a>
<a href="prices/freshwater-fish-prices.html">Price list</a>
<a href="../index.html">back to index</a>
<a href="https://www.thenetninja.co.uk/" target="_blank">Try out some awesome web tutorials</a>
<a href="downloads/all-prices.pdf">Download our full price list</a>
<div id="top"></div>
<a href="#top">to top</a>

11 – HTML Lists

Types of Lists

  • Unordered list – <ul> + <li>
  • Ordered list – <ol> + <li>
  • Definition list – <dl> + <dt> + <dd>
<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>
<ol>
  <li>Step one</li>
  <li>Step two
    <ol>
      <li>Step 2.1</li>
      <li>Step 2.2</li>
  </li>
  <li>Step three</li>
</ol>
<dl>
  <dt>Unordered list</dt>
  <dd>Unordered list is a list without any consideration to order</dd>
  <dt>Ordered list</dt>
  <dd>This is a list in which order is important</dd>
  <dt>Definition list</dt>
  <dd>Definition list contains terms and descriptions, much like a dictionary</dd>
</dl>

13 – The div Tag

The div tag explained

  • div stands for ‘division’
  • Used to divide content into logical sections
<div></div>

14 – ID’s and Classes

  • An ID is a UNIQUE identifier, used once on a page.
    • E.g. header, about
  • A Class is an identifier of an element which can be used multiple times on a page
    • E.g. Comment (when there are multiple comments)
  • Both are used for JS and CSS
<div id="header></div>
<div id="navigation></div>
<div id="types-of-fish></div>
<div id="about-us"></div>
<div id="contact-us"></div>

<div class="contact-method"></div>

<div id="comments"></div>
<div class="single-comment"></div>

15 – Adding CSS to HTML

3 Ways to Add CSS

  1. Inline styling using the style attribute
  2. Within <style> tags in the head of the document
  3. By linking up a stylesheet in the head
<a style="color: red;"></a>
<head>
  <style>
    #navigation a{
      color: red;
    }
  </style>
</head>
<head>
  <link rel="stylesheet" type="text/css" href="css/main.css>
</head>

h1{
  color: blue;
  text-decoration: underline;
  text-transform: uppercase;
}

16 – Adding JavaScript to HTML

How To Add JavaScript

  • In the <script> tag in the head or body
  • By linking a .js file in the head or body
  • Inline (not a good practise)
<head>
  <script>
    alert("Yo Ninja, welcome to my website!");
  </script>
</head>
<body>
  <script>
    alert("Yo Ninja, welcome to my website!");
  </script>
</body>
<head>
  <script src="scripts/main.js"></script>
</head>

alert("Yo Ninja, welcome to my website!");
<a onclick="alert('Yo Ninja, welcome to my website!');"></a>
